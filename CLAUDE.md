# FraudGuard — project rules

## Commits
- **Do NOT** add `Co-Authored-By: Claude <noreply@anthropic.com>` or any other
  Claude/anthropic attribution trailer to commit messages.
- **Do NOT** add `🤖 Generated with [Claude Code]` or similar footers to commit
  messages or PR bodies.
- Keep the commit subject ≤ 72 chars, body wrapped at 72 cols, no trailers.

## Pull requests
- PR body may end with a short bullet list summarizing changes, but no Claude
  attribution, no `🤖 Generated with…` line, no co-author trailer.
